About Me

I'm currently a senior at Carnegie Mellon University and am interested in software development and computer engineering. I'm majoring in Electrical and Computer Engineering, and minoring in Computer Science.

This past summer, I worked as a Software Engineering Intern at Salesforce, on the cloud infrastructure team building a capacity dashboard in Java to help back-end infrastructure planning. In the summer of 2014, I interned at IBM, building a library in Python for private cloud users to migrate easily to OpenStack, an open source cloud computing platform. Currently I'm working on a research project with Professor Pulkit Grover, in building an Android app to help blind individuals navigate using echolocation.

In my free time, I enjoy playing and watching soccer, and music.
